# Programador Web Nivel 2

## Acerca del Curso

Con el curso de Programador Web podrás realizar páginas y aplicaciones web interactivas usando las siguientes tecnologias: **Javascript**, **JQuery**, **AJAX** y **Metodología SCRUM**.
